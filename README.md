# SHDA Management App

A nonprofit management app for:
- Members
- Donations
- Expenses
- Reports

## Features
- Offline-first (SQLite)
- Firebase sync
- PDF reports
- GitHub auto APK build

## How to Build APK
1. Push code to GitHub
2. Go to Actions
3. Download APK from Artifacts
